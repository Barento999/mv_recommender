================================================================================
                      MODEL ARCHITECTURE
================================================================================

Abstract Base Class (MLModel):
  - train() → bool
  - recommend() → List[(movie_id, score)]
  - get_metrics() → Dict

Implementation (UserUserCollaborativeFiltering):
  - User-item rating matrix (sparse)
  - Cosine similarity between users
  - K-NN prediction (k=10)
  - Weighted average scoring
  - Cold start handling

Algorithm:
  1. Build matrix from ratings
  2. Compute similarities
  3. Find k similar users
  4. Predict unrated movie scores
  5. Return top-N by score

Performance:
  - Train: 0.3-0.5s for 2000 movies
  - Recommend: 50-100ms per user
  - Memory: ~50-100MB for model
  - Scalable to 10,000+ users
